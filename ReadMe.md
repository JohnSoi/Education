## Инструкция

Для удобства навигации можно воспользоваться программой [Obsidian](https://obsidian.md/).

Список необходимого ПО, а так же ссылки для их скачивания приводятся в файле ```Введение.md``` каждого раздела. 

Все материалы сопровождаются примерами и тестовыми заданиями для самопроверки. 
Ответы на них даются под спойлером сразу после текста.

Материалы с префиксом ```(Дополнительно)``` в начале файла необязательны к обучению и дают дополнительные знания.

Внутри материалов так же присутствуют ссылки на другие статьи или сторонние материалы.
